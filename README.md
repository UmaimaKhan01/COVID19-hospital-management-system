- Frontend
	- HTML5
	- CSS3
	- JQuery
    - Bootstrap4
- Backend
    - Django framework
- Database
    - SQLite

### Local Setup
- Clone repository.
- Setup virtual environment
- Exceute `pip install -r requirements.txt`.
- run `python manage.py runserver`.
- Go to the address shown in your cmd and copy it in your web browser.
